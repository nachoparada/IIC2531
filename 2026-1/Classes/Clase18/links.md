# OWASP Top 10 - 2021

Las 10 vulnerabilidades de seguridad más críticas en aplicaciones web según OWASP (Open Web Application Security Project).

## Las 10 principales vulnerabilidades

1. **Broken Access Control** — Fallas en restricciones de lo que usuarios autenticados pueden hacer
   - https://owasp.org/Top10/A01_2021-Broken_Access_Control/

2. **Cryptographic Failures** — Fallas relacionadas con criptografía que llevan a exposición de datos sensibles
   - https://owasp.org/Top10/A02_2021-Cryptographic_Failures/

3. **Injection** — SQL, NoSQL, OS, LDAP injection cuando datos no confiables se envían a un intérprete
   - https://owasp.org/Top10/A03_2021-Injection/

4. **Insecure Design** — Fallas de diseño y arquitectura, diferente de fallas de implementación
   - https://owasp.org/Top10/A04_2021-Insecure_Design/

5. **Security Misconfiguration** — Configuraciones de seguridad incorrectas o ausentes
   - https://owasp.org/Top10/A05_2021-Security_Misconfiguration/

6. **Vulnerable and Outdated Components** — Uso de componentes con vulnerabilidades conocidas
   - https://owasp.org/Top10/A06_2021-Vulnerable_and_Outdated_Components/

7. **Identification and Authentication Failures** — Fallas en confirmación de identidad del usuario
   - https://owasp.org/Top10/A07_2021-Identification_and_Authentication_Failures/

8. **Software and Data Integrity Failures** — Fallas relacionadas con código e infraestructura que no protege contra violaciones de integridad
   - https://owasp.org/Top10/A08_2021-Software_and_Data_Integrity_Failures/

9. **Security Logging and Monitoring Failures** — Fallas en detección, escalamiento y respuesta a brechas activas
   - https://owasp.org/Top10/A09_2021-Security_Logging_and_Monitoring_Failures/

10. **Server-Side Request Forgery (SSRF)** — Fallas cuando una aplicación web obtiene un recurso remoto sin validar la URL proporcionada por el usuario
    - https://owasp.org/Top10/A10_2021-Server-Side_Request_Forgery_%28SSRF%29/
